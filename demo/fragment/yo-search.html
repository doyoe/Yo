例一效果（input聚焦时收缩）：

取消

例一代码演示：

<form class="yo-search"> <label class="action"> <span class="yo-ico">&#xf067;</span> <input type="text" class="input input-shrink" placeholder="输入搜索关键字..." /> </label> <span class="cancel">取消</span> </form>

例二效果（input聚焦时不收缩）：

例二代码演示：

<form class="yo-search"> <label class="action"> <span class="yo-ico">&#xf067;</span> <input type="text" class="input" placeholder="输入搜索关键字..." /> </label> </form>

只需移除[input-shrink]类名

<input class="input input-shrink" ... /> ↓↓↓ <input class="input" ... />

例三效果（带搜索按钮的搜索）：

搜索

例三代码演示：

<form class="yo-search"> <label class="action"> <span class="yo-ico">&#xf067;</span> <input type="text" class="input" placeholder="输入搜索关键字..." /> </label> <span class="yo-btn">搜索</span> <!-- 增加这行即可 --> </form>
