示例描述：

<ul class="yo-rating"> <li class="item"><span>1分</span></li> <li class="item"><span>2分</span></li> <li class="item"><span>3分</span></li> <li class="item"><span>4分</span></li> <li class="item"><span>5分</span></li> </ul>

假设你自定义打分效果，只需这样扩展：

@include yo-rating( // 自定义扩展的名字 $name: test, // 单个子项的宽度 $item-width: .2rem, // 单个子项的高度 $item-height: .2rem, // 子项间间隙 $item-psace: .1rem, // 打分图像 $url: "new-star.png" );

使用方法：

<ul class="yo-rating yo-rating-test">...</ul>

如果只是纯展示，你需要这样：

<ul class="yo-rating">...</ul> ↓↓↓ <ul class="yo-rating yo-rating-readonly">...</ul>
